    structure: "Subordinating conj + Clause + Comma + Main Clause (or reverse)"
    example: "Although the team worked overtime, the project was delayed."
  - label: "Correlative Conjunctions"
    structure: "both...and / either...or / neither...nor / not only...but also"
    example: "Either the director or the assistant manager will attend the conference."

  - eng: "The conference was cancelled because the venue had not been confirmed."

  - eng: "Although profits were up, the board decided to postpone the expansion."
    vie: "Mặc dù lợi nhuận tăng, ban giám đốc vẫn quyết định hoãn mở rộng."
    note: "'Although' introduces contrast. 'Although' cannot be used with 'but' in the same clause."
commonMistakes:
  - mistake: "Although the project was successful, but we still need to improve the process."
    correction: "Although the project was successful, we still need to improve the process."
    explanation: "'Although' and 'but' are both conjunctions expressing contrast. Never use both together."
  - mistake: "Because of the traffic, so the meeting started late."
    correction: "Because of the traffic, the meeting started late."
    explanation: "'Because of' and 'so' cannot both be used in the same sentence. Use one or the other."
coordinatingConjunctions:
  - conjunction: "for"
    function: "reason"
    example: "He resigned, for he had found a better opportunity."
  - conjunction: "and"
    function: "addition"
    example: "She reviewed the report and sent it to the director."
  - conjunction: "nor"
    function: "negative addition"
    example: "He didn't call, nor did he send an email."
  - conjunction: "but"
    function: "contrast"
    example: "The plan was approved, but the budget was cut."
  - conjunction: "or"
    function: "alternative"
    example: "You can submit the form online or in person."
  - conjunction: "yet"
    function: "contrast (formal)"
    example: "The results were unexpected, yet the team remained confident."
  - conjunction: "so"
    function: "result"
    example: "The server crashed, so the meeting was postponed."
memoryTips:
  - "FANBOYS = For, And, Nor, But, Or, Yet, So (7 coordinating conjunctions)."
  - "Never combine: although + but, because + so, despite + but. Use ONLY ONE."
  - "Correlative conjunctions must maintain parallel structure: Both X and Y (same grammatical form)."
cheatSheet:
  - "Reason: because (clause), because of (noun phrase), since, as"
  - "Contrast: although, even though, though, while, whereas, however (+ comma)"
  - "Condition: if, unless, as long as, provided that"
  - "Result: so, therefore, thus, as a result"
  - "Trap: although + but = WRONG | because + so = WRONG"
summary: "Use coordinating conjunctions to join equal clauses. Use subordinating conjunctions for dependent clauses. Never double-up contrasting conjunctions."

### Conjunction Categories

| Category | Examples | Use |
|----------|----------|-----|
| **Coordinating (FANBOYS)** | for, and, nor, but, or, yet, so | Join equal clauses |
| **Subordinating (Reason)** | because, since, as | Introduce reason clause |
| **Subordinating (Contrast)** | although, even though, though, while | Introduce contrast clause |
| **Subordinating (Condition)** | if, unless, as long as | Introduce condition clause |
| **Correlative** | both...and, either...or, neither...nor | Pair equal elements |

### Critical TOEIC Grammar Rule
Never combine two conjunctions that express the same relationship:

❌ **Although** she worked hard, **but** she failed.
✅ **Although** she worked hard, she failed.
✅ She worked hard, **but** she failed.

❌ **Because** the data was wrong, **so** the report was rejected.
✅ **Because** the data was wrong, the report was rejected.
✅ The data was wrong, **so** the report was rejected.
