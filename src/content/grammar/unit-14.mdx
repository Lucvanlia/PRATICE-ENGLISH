### Conditional Types Summary Table

| Type | Probability | If-Clause | Main Clause | Example |
|------|-------------|-----------|-------------|---------|
| **0** | Always true | Present Simple | Present Simple | If you mix colours, you get new colours. |
| **1** | Possible | Present Simple | will + V1 | If she calls, I'll respond. |
| **2** | Hypothetical | Past Simple | would + V1 | If I ran the company, I'd change this. |
| **3** | Impossible | Past Perfect | would have + V3 | If they'd planned better, they'd have succeeded. |

### Mixed Conditionals
Sometimes Type 2 and Type 3 mix:
> If I **had taken** (Type 3 - past) the course, I **would be** (Type 2 - present) more qualified now.
(Past unfulfilled condition → present unreal result)
